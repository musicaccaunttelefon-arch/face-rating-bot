# 🤖 Бот оценки внешности (без AI API)

Бот анализирует лицо через **MediaPipe Face Mesh** — 468 точек лица,
и считает 5 геометрических метрик:

| Метрика | Вес | Что измеряет |
|---|---|---|
| Симметрия лица | 35% | Насколько левая и правая стороны совпадают |
| Золотое сечение | 20% | Соотношение высоты и ширины лица (идеал 1.618) |
| Расположение глаз | 20% | Межглазное расстояние vs ширина глаза |
| Правило третей | 15% | Лоб : нос : подбородок = 1:1:1 |
| Пропорции рта | 10% | Ширина рта vs расстояние между глаз |

---

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

**Windows:**
```powershell
$env:TELEGRAM_TOKEN="твой_токен"
python bot.py
```

**Linux/Mac:**
```bash
export TELEGRAM_TOKEN="твой_токен"
python bot.py
```

**Или вставь токен прямо в bot.py** в строку:
```python
TELEGRAM_TOKEN = "твой_токен_сюда"
```

Токен получи у @BotFather → /newbot
